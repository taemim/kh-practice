if문/삼항연산자

prompt로 "당신의 등급은 무엇입니까?" 라는 질문의 답을 받고 (기본 값 : 손님)
"손님"의 경우 "글 읽기",
"일반 회원"의 경우 "글 읽기, 글 쓰기",
"운영자"의 경우 "글 읽기, 글 쓰기, 글 삭제"
라는 값을 변수에 저장하여 아래와 같이 alert로 출력한다.
Ex. 운영자의 경우 글 읽기, 글 쓰기, 글 삭제가 가능합니다.
if문 사용 시, 삼항 연산자 사용 시의 코드를 각각 별도의 함수로 선언하여 내부에 작성하고 호출하여 수행 테스트를 한다.

반복문

"반드시 짝수를 입력하세요" 라는 prompt 출력 후 사용자가 짝수가 아닌 값을 입력한 경우 반복문을 사용해 동일한 prompt 창 출력한다. 사용자가 짝수를 입력하거나 취소 버튼을 누른 경우, 혹은 아무것도 입력하지 않고 확인 버튼을 누른 경우엔 prompt 창 출력을 반복하지 않고 입력 받은 값 console로 출력한다.
